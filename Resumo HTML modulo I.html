Resumo módulo I de HTML

Trabalhando com paragráfos e quebra de linha

Para criar um paragráfo, utilize a tag <p>

Você pode criar vários paragráfos seguidos utilizando essa tag.
Além disso, para fazer uma quebra de linha, pode-se utilizar a tag <br>

Trabalhando com simbolos e emojis

Para utilizar simbolos e emojis, é necessário que você busque a codificação do simbolo ou do emoji para implementá-lo.
Deve-se começar com o E comercial ''&'' antes do código especifico do simbolo/emoji.
Por exemplo:
© ™ ✌

Trabalhando com imagens

Para utilizar imagens, você deve usar a tag:
<img src=''## Nome da imagem ##'' alt=''## Texto alternativo da imagem ##'' width=''## Largura da imagem ##'' height=''## Altura da imagem ## ''

Trabalhando com formatações

É possivel trabalhar com palavras em negrito, italico e outros tipos de formatação.
Para tal, utilize as seguintes tags: 

    
        <strong>              para negritar alguma palavra ou texto, denotando uma forte importância
        <EM>                  para textos italicos, denotando do resto do conteudo por algum motivo
        <mark>                para marcar algum conteudo - ''Grifando-o''
        <small>               para configurar um texto ou palavra em um tamanho menor
        <del>                 para configurar um texto deletado
        <ins>                 para configuar um texto inserido
        <sup>                 para obter um texto sobrescrito, tipo: 2x2
        <sub>                 para obter um texto subscrito, tipo: H2O

Trabalhando com citações e códigos

É possivel utilizar as seguintes tags: 

        
        <code>               para utilizar a formatação de código-fonte     
        <q>                  para utilizar citações simples: O computador é um burro muito rápido 
        <blockquote>         para citações completas 
        <abbr>               para abreviações, onde você pode passar o mouse, tipo aqui
        <bdo dir=''rtl''>            para textos invertidos, entendeu? 😜

Trabalhando com listas e definições

Listas ordenadas e não ordenadas

Para utilizar listas ordenadas, tipo assim:

  1. Acordar
  2. Tomar café da manhã
  3. Estudar

É necessário utilizar as seguintes tags:

        
         <ol type=''1'' start=''0''>  Ou seja, a tag <ol> para abrir uma lista ordenada, em seguida de type=''1'' e start=''0'' para identificar o tipo e seu início.
                                      Não se esqueça de colocar <li> sempre, para começar a listar.

Listas não ordenadas

Agora para utilizar listas não ordenadas, tipo assim:

  • Pão
  • Azeite
  • Leite

É necessário utilizar as seguites tags:

        
         <ul type=''## Tipo de simbolo ##'' Ou seja, a tag <ul> para abrir uma lista não ordenada, em seguida de type=''####'' identificar o tipo, por exemplo ''square''
                                        Não se esqueça de colocar <li> sempre, para começar a listar.

Trabalhando com Imagens dinâmicas

É possivel utilizar imagens dinâmicas no seu site, permitindo que elas se adaptem sozinhas dependendo do tamanho da tela do usuário

Para tal, utilize a seguite tag:

    
        <picture>
        <source media="(max-width: ## tamanho desejado: 750px ##)" srcset="## diretorio da imagem ##" type="## imagem menor ##" >
        </picture>

É possível fazer isso para por exemplo três tamanhos diferentes de imagem: pequena, média e grande.

Apenas se atente ao fato de que é preciso colocar as tags em ''ordem de tamanho'', ou seja, a tag com a menor imagem acima das outras.

Trabalhando com audios e videos

É possivel trabalhar com audios no seu site, vou dar um exemplo

Para tal, utilize a seguite tag:

    
        <audio preload="metadata" autoplay controls>
        <source src="## Endereço do seu áudio ##" type="audio/mpeg">
        </audio>

Para hospedar um video no seu site, a forma mais econômica e simples é utilizar videos do youtube, pois irá poupar o consumo de dados do seu site

Dessa forma, é simples: Basta acessar o Youtube, procurar um video desejado, clicar em "Compartilhar", e o próprio youtube irá ter a opção de "incorporar", ai basta colar!
